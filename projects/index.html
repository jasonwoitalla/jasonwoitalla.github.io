The goals for this project are to provide functionality and be aesthetically pleasing. The functionality comes from the components being attached to a lazy Susan assembly. Thermometer Display

A project for my machine architecture class in my sophomore year of college. This project required to read input from a sensor and give a bitwise output to display thermometer temperatures to a display. The program is to run on a low-level computer chip incapable of floating-point types and division. I’m happy to share the source code located on a private GitHub repository.
